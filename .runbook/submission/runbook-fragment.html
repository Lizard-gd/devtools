# Правило: Перед push: git status clean

## Где проверять
### В IDE (IntelliJ IDEA)
- Открыть Commit Tool Window (Alt+0 / Cmd+0)
- Проверить, что списки Unversioned Files и Changes пусты

### В терминале
git status
Должно быть: "nothing to commit, working tree clean"

## Текущие правила игнорирования (.gitignore)
# Gradle Wrapper
.gradle/
build/
!gradle/wrapper/gradle-wrapper.jar
!gradle/wrapper/gradle-wrapper.properties

# IntelliJ IDEA
.idea/
*.iml
out/

# OS files
.DS_Store
Thumbs.db

## Если статус не чистый
1. Проверить, должен ли файл быть в репозитории
2. Если нет - добавить паттерн в .gitignore
3. Удалить из индекса: git rm --cached 
4. Закоммитить изменения
